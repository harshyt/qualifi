# UI/UX Design Guidelines: ResumeScreener (High-Clarity Light)

## 1. Design Philosophy

- **Light & Airy:** Prioritize white space. Use thin borders (#E0E0E0) instead of shadows to define containers.
- **Aesthetic:** Minimalist Material 3. The app should feel like a modern SaaS (e.g., Notion/Linear).
- **Tone:** Professional, calm, and efficient.

## 2. Color Palette (Hex Codes)

- **Primary:** #2196F3 (Soft Azure) - Actions, active states, and primary buttons.
- **Background:** #F9FAFB (Cloud White) - Main page background.
- **Surface:** #FFFFFF (Pure White) - Cards, table rows, and modals.
- **Text Primary:** #37474F (Blue-Gray) - For headings and body text.
- **Text Secondary:** #78909C (Muted Gray) - For labels and metadata.
- **Accents:** - Success: #E8F5E9 (Pale Mint background) / #2E7D32 (Dark Green text).
  - Warning: #FFF3E0 (Pale Peach background) / #E65100 (Deep Orange text).

## 3. Typography & Components

- **Font:** Inter or System Sans-Serif.
- **Buttons:** - Primary: Solid Soft Azure with White text. No shadows (`disableElevation`).
  - Secondary: Outlined with Soft Azure border and text.
- **Tables:** No vertical lines. Use thin horizontal dividers only.
- **Input Fields:** Outlined style with 4px border-radius.
